# Pengelolaan Anggota UKM POLIBAN  Aplikasi manajemen data untuk mengatasi pencatatan anggota UKM yang tidak terorganisir. Dikelola oleh 1 Admin, sistem ini mendukung pendaftaran (Mahasiswa, UKM, Anggota), CRUD, pencarian, cetak, &amp; export Excel. Aturan: 1 mahasiswa terdaftar resmi hanya bisa ikut 1 UKM.
